# 事前作業
* [CentOS6系にremiレポジトリを追加する。](https://lighthouse-dev.github.io/remi-releaseinstall/)

<br>

# 確認

	$ php -v  
	-bash: php: command not found  



	$ rpm -qa | grep php  
	$ rpm -qa | grep remi-release  
	remi-release-6.8-1.el6.remi.noarch
